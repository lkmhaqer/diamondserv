Guide to creating a server

  • Go to Profile > Add Minecraft User
  • Add your minecraft username, and any user you want to OP or add to an allow-list
  • Use the Create Server tab
    • Name: A unique server name, can contain letters, numbers and hypens.
    • Host: The linux computer where your server will live.
    • Game Type: Creative is a free play environment, Survivial is competitive PvE game.
    • Server Type: Vanilla, FeedTheBeast, Forge, pick your mine flavor.
      • Note:The version number on modded types is the modpack version.
    • Op List: Select multiple names (added in your profile) to have op power on the server
  • Click Create Server
  • You will be taken to the server detail page.
  • Refresh this page until you see the "server started" text at the bottom of the log.
  • Vanilla servers take about 60 seconds to boot, Modded servers are 5 to 10 minutes.
